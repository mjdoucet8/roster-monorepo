# prompts

Use this folder for reusable prompts, agent instructions, and task templates that are valuable enough to keep.

A prompt file should explain what the prompt is for, when to use it, what inputs it expects, and any known caveats. If the prompt produces a specific artifact, include the expected output shape.

Create a prompt when an instruction has been tuned through repeated use, captures a project-specific workflow, or helps agents produce consistent results.
